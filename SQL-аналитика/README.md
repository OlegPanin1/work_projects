# SQL-аналитика: возвраты клиентов и атрибуция заказов

## Контекст
Набор SQL-запросов для аналитики эффективности рекламных кампаний в e-commerce / advertising network. Запросы отражают реальные бизнес-задачи, с которыми сталкивался в работе.

## Запросы

### 1. Анализ возвратов и оттока клиентов
**Файл:** `queries_affiliate_analytics.sql` (основной запрос)

**Бизнес-задача:** определить группы возвратов клиентов (30/60/90 дней) для расчёта LTV в разрезе когорт лояльности и оценки эффективности удерживающих кампаний.

**Логика:**
- Для каждого заказа вычисляется дата предыдущего заказа (LAG)
- Флаги `is_return_N` показывают, был ли перерыв между заказами больше N дней
- `return_group` — кумулятивная сумма флагов, определяющая "номер жизни" клиента
- Итоговая выборка агрегирует заказы по дням, сегментам и каналам

**Ключевые конструкции:**
- Оконные функции: LAG, SUM OVER с ROWS BETWEEN
- CTE для поэтапного построения логики
- LEFT JOIN с подзапросами
- Работа с интервалами (INTERVAL)

### 2. Юнит-экономика по каналам
**Бизнес-задача:** расчёт ROMI, CPA, LTV в разрезе каналов привлечения для оптимизации медиаплана.

**Метрики:**
- CPA (Cost Per Action) — стоимость привлечения
- ROMI (Return on Marketing Investment) — окупаемость инвестиций
- LTV (Lifetime Value) — пожизненная ценность клиента

### 3. Воронка конверсии
**Бизнес-задача:** выявление точек падения на пути клика → регистрация → депозит по каналам привлечения.

### 4. Анализ креативов
**Бизнес-задача:** определение топ-10 креативов по ROMI для оптимизации креативной стратегии в performance-кампаниях.

### 5. Сравнение моделей атрибуции
**Бизнес-задача:** сопоставление распределения конверсий между last click и first click моделями атрибуции.

## Используемые SQL-конструкции
| Конструкция | Где используется |
|-------------|------------------|
| Оконные функции (LAG, FIRST_VALUE, LAST_VALUE, SUM OVER) | Анализ возвратов, атрибуция |
| CTE (WITH) | Все запросы — для читаемости |
| LEFT JOIN с подзапросами | Привязка атрибуции, справочников |
| CASE с интервалами | Флаги возвратов |
| Агрегации + GROUP BY | Сводные метрики |
| NULLIF | Защита от деления на ноль |
