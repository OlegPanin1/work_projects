# A/B тест: сравнение моделей атрибуции

## Контекст
В рамках работы аналитиком в advertising network проводил A/B-тест для оценки влияния новой модели атрибуции на распределение рекламного бюджета и эффективность привлечения.

## Цель
Проверить, приведёт ли переход от last click к multi-touch модели к увеличению конверсии в целевое действие (регистрация + депозит) без роста стоимости привлечения.

## Дизайн теста
| Параметр | Значение |
|----------|---------|
| Контроль (А) | last click — вся ценность конверсии присваивается последнему клику |
| Тест (B) | multi-touch (линейная модель) — вес распределяется равномерно между всеми касаниями |
| Каналы | Meta, Google, TikTok |
| Длительность | 14 дней |
| Выборка | 10 000 пользователей (5 000 в группе) |

## Результаты

### 1. Основной результат
- Контрольная группа (А): конверсия **11.56%**
- Тестовая группа (B): конверсия **14.36%**
- **Прирост: +2.8 п.п. (+24% относительно контроля)**

### 2. Статистическая значимость
- 95% доверительный интервал разницы: **[1.5%; 4.1%]** (не пересекает ноль)
- p-value = 0.0001

Вывод: результат статистически значим, вероятность случайного эффекта менее 0.01%.

### 3. Экономический эффект
При фиксированном бюджете:
- CPA снизился на **19%**
- ROMI вырос на **31 п.п.**
- Дополнительная прибыль при масштабировании на весь трафик: **+24% к объёму конверсий**

### 4. Разрез по каналам
| Канал | Контроль (CR) | Тест (CR) | Δ |
|-------|---------------|-----------|---|
| Meta  | 10.2%         | 13.1%     | +2.9 п.п. |
| Google| 12.8%         | 15.4%     | +2.6 п.п. |
| TikTok| 11.1%         | 14.2%     | +3.1 п.п. |

Эффект равномерный по всем каналам — искажения отсутствуют.

### 5. Ограничения теста
- Длительность 14 дней не позволяет оценить долгосрочный LTV пользователей
- Тест проводился только на платных каналах, влияние на органический трафик не оценивалось

## Рекомендация
Внедрить multi-touch модель атрибуции на весь поток платного трафика. Рекомендуется провести повторный анализ через 30–60 дней для оценки влияния на LTV.
