# Когортный анализ retention и LTV

## Контекст
Оценка удержания пользователей и динамики LTV (Lifetime Value) для оптимизации маркетинговой стратегии и распределения бюджета между каналами привлечения.

## Задача
1. Оценить retention пользователей в разрезе когорт по месяцам
2. Рассчитать кумулятивный LTV для каждой когорты
3. Сравнить LTV по каналам привлечения
4. Сформулировать рекомендации по оптимизации бюджета

## Данные
- Период: январь 2023 — март 2024
- Объём: 10 000 пользователей
- Каналы привлечения: organic, fb_ads, google_ads, referral

## Методология
- **Когортный анализ** — группировка пользователей по месяцу регистрации
- **Retention** — процент пользователей, совершивших покупку в каждом месяце после регистрации
- **LTV** — кумулятивная выручка на пользователя с учётом времени жизни

## Результаты

### Retention
| Tenure | 1 месяц | 3 месяца | 6 месяцев |
|--------|---------|----------|-----------|
| Retention | 86–92% | 48–54% | 13–18% |

Наиболее стабильное удержание у когорт начала 2023 года (январь–март), что коррелирует с запуском программы лояльности в Q1.

### LTV по каналам (12 months)
| Канал | 12-month LTV |
|-------|--------------|
| Referral | $4,442 |
| Google Ads | $4,177 |
| FB Ads | $4,089 |
| Organic | $3,825 |

### Динамика LTV по когортам
Когорты 2023 года показывают устойчивый рост LTV в первые 6 месяцев с последующей стабилизацией. Ранние когорты демонстрируют более высокий LTV на дальних горизонтах, что характерно для зрелого продукта.

## Рекомендации
1. **Увеличить бюджет на referral-программу** — показывает максимальный LTV ($4,442) при минимальных затратах на привлечение
2. **Пересмотреть стратегию по organic-трафику** — LTV на 14% ниже лидера, требуется анализ качества трафика и точек входа
3. **Google Ads vs FB Ads** — при сопоставимом LTV ($4,177 vs $4,089) принять решение на основе CAC: если CAC на FB Ads выше, перераспределить бюджет в пользу Google
4. **Анализ когорт начала 2023** — выявить факторы высокого удержания и масштабировать на новые когорты

## Ограничения анализа
- Горизонт анализа ограничен 12 месяцами для поздних когорт
- Не учитывается сезонность внутри года
- LTV рассчитан без учёта маржинальности
